# Chart Assets

This directory contains template files and specifications for generating professional financial charts.

## Chart Templates

### ratio_analysis_template.png
**Purpose**: Template structure for ratio analysis charts
**Specifications**:
- Dimensions: 1200x800 pixels (16:10 ratio)
- Format: PNG with 300 DPI resolution
- Colors: Professional blue/green/red color scheme
- Gridlines: Light gray, horizontal only
- Legend: Positioned at top or right

**Chart Structure**:
- Title area for company name and chart type
- X-axis for time periods (quarters/years)
- Y-axis for ratio values with appropriate scaling
- Multiple line series for different ratios
- Reference lines for industry benchmarks

### trend_analysis_template.png
**Purpose**: Template structure for financial trend charts
**Specifications**:
- Dimensions: 1200x800 pixels (16:10 ratio)
- Format: PNG with 300 DPI resolution
- Chart Type: Dual-axis line chart
- Colors: Revenue (blue), Earnings (red)

**Chart Structure**:
- Primary Y-axis: Revenue (left, currency format)
- Secondary Y-axis: Earnings (right, currency format)
- X-axis: Time periods with consistent intervals
- Line styles: Solid for revenue, dashed for earnings
- Annotations: Key events and growth rates

### waterfall_template.png
**Purpose**: Template structure for cash flow waterfall charts
**Specifications**:
- Dimensions: 1000x700 pixels
- Format: PNG with 300 DPI resolution
- Colors: Green for inflows, red for outflows
- Connector lines: Black lines between bars

**Chart Structure**:
- Starting point: Net income or operating cash flow
- Positive values: Cash inflows (green bars)
- Negative values: Cash outflows (red bars)
- Ending point: Free cash flow or ending cash balance
- Value labels: On each bar with proper formatting

## Chart Generation Specifications

### Color Standards
**Primary Colors**:
- Liquidity Ratios: Blue family (#2E86AB, #5AA9E6)
- Profitability Ratios: Green family (#2CA453, #66BB6A)
- Leverage Ratios: Red/Orange family (#D32F2F, #FF7043)
- Efficiency Ratios: Purple family (#7E57C2, #AB47BC)

**Accessibility**:
- Ensure color contrast ratios meet WCAG standards
- Use patterns or textures for color-blind accessibility
- Avoid red-green combinations alone
- Provide alternative text descriptions

### Typography Standards
**Font Specifications**:
- Font Family: Sans-serif (Arial, Helvetica, system fonts)
- Title: 14-16pt, bold, centered
- Axis Labels: 11-12pt, readable
- Data Labels: 9-10pt, clear
- Legend: 10-11pt, positioned appropriately

**Text Formatting**:
- Currency: Proper symbol and thousand separators
- Percentages: With % symbol, appropriate rounding
- Dates: Consistent format (MMM YYYY or YYYY)
- Numbers: Appropriate decimal places

### Grid and Layout Standards
**Gridlines**:
- Color: Light gray (#E0E0E0)
- Style: Thin lines, horizontal only for line charts
- Opacity: Subtle, not distracting from data

**Margins and Spacing**:
- Chart margins: Minimum 20px around chart area
- Spacing: Consistent between chart elements
- Alignment: Proper alignment of axes and labels
- Padding: Adequate padding for readability

### Data Presentation Standards
**Precision and Formatting**:
- Currency: Use appropriate symbols and formatting
- Percentages: Display with % symbol, proper rounding
- Ratios: Appropriate decimal places (usually 2-3)
- Large numbers: Use K, M, B abbreviations when appropriate

**Annotations**:
- Data labels: On key data points
- Trend lines: When showing direction
- Reference lines: For benchmarks or targets
- Callouts: For significant events or changes

## Chart Integration Guidelines

### PowerPoint Integration
**Requirements**:
- Resolution: 300 DPI for professional presentation
- File format: PNG or embedded chart objects
- Size: Optimized for slide dimensions
- Animation: Use sparingly, only when it adds value

**Best Practices**:
- Maintain consistent chart styles across slides
- Use chart titles that complement slide titles
- Ensure charts are readable from presentation distance
- Include data source and date information

### Word Document Integration
**Requirements**:
- Resolution: 150-300 DPI for print quality
- File format: High-resolution PNG or embedded charts
- Size: Fit within document margins
- Captions: Include descriptive captions

**Best Practices**:
- Charts should be readable when printed
- Use consistent caption format
- Reference charts in document text
- Maintain consistent chart styles

### Excel Integration
**Requirements**:
- Chart objects: Use Excel chart objects for interactivity
- Data links: Maintain live data connections
- Formatting: Apply consistent Excel chart styles
- Export: Export as high-resolution images when needed

**Best Practices**:
- Charts should update automatically with data changes
- Use Excel's built-in chart formatting tools
- Maintain consistent color schemes
- Include proper chart titles and axis labels
