# Template Structure Documentation

This document describes the structure and content for the report templates used in the SEC Filings Analysis skill.

## Analysis Report PowerPoint Template Structure

### Slide 1: Title Slide
**Layout**: Title slide layout
**Content**:
- Company Name (Large, Centered)
- "Financial Analysis Report" (Subtitle)
- Date (Bottom Right)
- Analyst Name (Bottom Left)

**Design Elements**:
- Company logo placeholder
- Professional color scheme
- Clean, minimalistic design

### Slide 2: Executive Summary
**Layout**: Title and content layout
**Content**:
- Overall investment rating
- Key financial highlights
- Major risks identified
- Primary recommendations

**Content Areas**:
- Text box for summary paragraph
- Bullet points for key metrics
- Risk level indicator

### Slide 3: Financial Highlights
**Layout**: Two-column layout
**Left Column**: Key Metrics
- Revenue growth (YoY)
- Net margin percentage
- EPS growth
- Free cash flow

**Right Column**: Financial Health Indicators
- Debt-to-equity ratio
- Current ratio
- ROE percentage
- Liquidity score

### Slide 4-5: Ratio Analysis
**Layout**: Chart layout
**Content**:
- Multi-line ratio trend charts
- Liquidity ratios over time
- Profitability ratios comparison
- Industry benchmarking

**Chart Requirements**:
- Professional color scheme
- Clear axis labels
- Legend for multiple lines
- Data source attribution

### Slide 6-7: Trend Analysis
**Layout**: Chart layout
**Content**:
- Revenue and earnings trends
- Margin progression over time
- Growth rate analysis
- Seasonal patterns

**Chart Types**:
- Dual-axis line charts
- Area charts for margin trends
- Bar charts for growth rates

### Slide 8: Risk Assessment
**Layout**: Title and content layout
**Content**:
- Risk level classification
- Top 3-5 identified risks
- Risk mitigation strategies
- Risk monitoring priorities

**Visual Elements**:
- Risk heat map
- Risk category distribution
- Mitigation progress indicators

### Slide 9: Valuation Analysis
**Layout**: Two-column layout
**Left Column**: Valuation Metrics
- P/E ratio comparison
- P/B ratio analysis
- DCF valuation summary
- Relative valuation

**Right Column**: Investment Thesis
- Key investment points
- Valuation rationale
- Upside/downside scenarios
- Investment timeline

### Slide 10: Recommendations
**Layout**: Title and content layout
**Content**:
- Investment recommendation
- Price target (if applicable)
- Risk-adjusted returns
- Next steps and monitoring

**Call to Action**:
- Specific recommendation
- Time horizon
- Monitoring criteria

## Template Implementation Notes

### Chart Integration
- All templates should support automatic chart insertion
- Chart placeholders should be clearly marked
- Charts should maintain consistent formatting
- Data source attribution should be included

### Content Population
- Templates should support automated content population
- Placeholder text should be clearly marked
- Formatting should be preserved during population
- Charts and tables should be dynamically generated

### Professional Standards
- All templates should maintain professional appearance
- Consistent color schemes and typography
- Proper spacing and alignment
- High-resolution image requirements

### Accessibility
- Color choices should consider color blindness
- Font sizes should be readable
- Contrast ratios should meet accessibility standards
- Alternative text for images and charts

These template structures provide the foundation for generating professional analysis reports and investment theses using the SEC Filings Analysis skill.
