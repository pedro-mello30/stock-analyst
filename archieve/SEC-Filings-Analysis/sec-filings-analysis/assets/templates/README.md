# Report Templates

This directory contains template files for generating professional analysis reports and investment theses.

## Template Files

### analysis_report.pptx
**Purpose**: 15-20 slide detailed analysis template
**Content Structure**:
- Title slide with company information
- Executive summary with key findings
- Financial highlights and performance metrics
- Ratio analysis with trend charts
- Trend analysis with revenue and margin trends
- Risk assessment and mitigation strategies
- Valuation metrics and comparables
- Investment recommendations

**Usage**: Use with `report_generator.py` for comprehensive analysis reports

### investment_thesis.docx
**Purpose**: 8-10 page comprehensive investment thesis template
**Content Structure**:
- Title page with company and thesis information
- Company overview and business model analysis
- Industry analysis and competitive positioning
- Financial analysis with detailed metrics
- Risk assessment and scenario analysis
- Valuation methodology and price targets
- Investment thesis and conclusion

**Usage**: Use with `report_generator.py` for detailed investment thesis documents

### executive_summary.pptx
**Purpose**: 5-7 slide high-level executive summary template
**Content Structure**:
- Title slide with company and date
- Key metrics and financial highlights
- Investment highlights and rationale
- Risk factors and mitigation
- Recommendation and next steps

**Usage**: Use with `report_generator.py` for executive-level summaries

## Template Customization

### Branding Integration
- Replace placeholder logos with company branding
- Update color schemes to match corporate colors
- Modify fonts to match brand guidelines
- Add company-specific disclaimers and legal text

### Content Customization
- Modify slide layouts for specific industries
- Add industry-specific analysis sections
- Customize chart types for different data
- Adjust report length based on requirements

### Technical Specifications
- Maintain minimum resolution of 300 DPI for print quality
- Use consistent font sizes and spacing
- Ensure proper image compression for file size optimization
- Maintain accessibility standards for color contrast

## Template Maintenance

### Version Control
- Track template changes with version numbers
- Maintain compatibility with report generator scripts
- Update templates when analysis frameworks change
- Document template modification history

### Quality Assurance
- Test templates with sample data before deployment
- Verify chart generation and formatting
- Check for broken links or missing elements
- Ensure consistent formatting across all templates

## Integration with Analysis Framework

2. **Data Integration**: Charts and tables are generated from processed financial data
3. **Consistency**: Templates maintain consistent formatting with analysis standards
